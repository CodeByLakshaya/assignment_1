# **Cursor Landing Page Clone**

A **static frontend recreation** of the Cursor (AI Code Editor) landing page built using **HTML5 and CSS3**.

---

## **Project Overview**

- **Type:** Static UI Clone  
- **Purpose:** Frontend layout and design practice  
- **Status:** Completed  

---

## **Sections Implemented**

- **Navigation Bar**
  - Logo and branding
  - Navigation links
  - Action buttons (**Sign in**, **Download**)

- **Hero Section**
  - Product description
  - **Primary Call-to-Action button**

- **Testimonials Section**
  - User feedback cards
  - Author details

- **Footer**
  - Product, Resources, Company, and Legal links

---

## **Typography**

- **Primary Font:** Segoe UI  
- **Fallback Fonts:** Tahoma, Geneva, Verdana, sans-serif  

---

## **Color Palette**

- **Background:** `#F7F7F4`
- **Primary Text:** `#000000`
- **Buttons / Footer:** `#14130F`
- **Card Background:** `#EDEDE9`

---

## **Technologies Used**

- **HTML5**
- **CSS3 (Flexbox & Grid)**

---

## **Limitations**

- **Desktop-only layout**
- **No JavaScript functionality**
- **Not fully responsive**
